
# Pizza Menu

A frontend project of pizza menu made using React Js.


## Run Locally

Clone the project

```bash
  git clone https://github.com/WaqarAhmad321/pizza-menu.git
```

Go to the project directory

```bash
  cd pizza-menu
```

Install dependencies

```bash
  npm create create-react-app
```

Start the server

```bash
  npm run start
```
